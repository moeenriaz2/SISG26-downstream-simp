Setup — Download PLINK & GCTA

Clone the SISG26-PRS repo and run setup.sh to install PLINK and GCTA, then add them to your PATH.

git clone https://github.com/moeenriaz2/SISG26-PRS.git
cd SISG26-PRS
bash setup.sh                      # downloads PLINK 1.9, PLINK 2, GCTB → exe/
export PATH="$PWD/exe:$PATH"

Part I — Simulate Data

Build a 1 Mb chromosome with 20 LD blocks and 100 known causal variants. Generate GWAS genotypes, phenotypes, and an LD reference panel.

Part II — Run COJO

Run GCTA-COJO with different LD reference sizes (N = 5000, 2000, 1000, 500) and heritability values. Measure power and false discovery rate.

Part III — Fix COJO?

Apply the --cojo-collinear 0.1 flag to reduce false positives when the LD reference is small.
